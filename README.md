# Fashion Store Sales Analysis Using SQL

## Overview

This project analyzes sales data from the online fashion store "LakuBanyak" using SQL and MySQL. The analysis focuses on product performance, sales trends, and business insights to support data-driven decision-making.

## Case Study

LakuBanyak is an online fashion store that sells clothing, shoes, bags, and accessories. This project aims to analyze sales data by:

1. Exploring product data.
2. Filtering products by category.
3. Calculating the total number of products.
4. Calculating total sales revenue.
5. Analyzing sales performance by product category.
6. Identifying the top-selling products.

## Key Findings

* The store offers 15 different products across multiple categories.
* Total sales revenue reached Rp 6,866,000.
* The Shoes category generated the highest sales revenue.
* Kaos Polos Hitam was the best-selling product.
* Popular products should be prioritized in inventory management and promotional activities.
